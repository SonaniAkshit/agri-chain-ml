# Agri-Chain: AI-Powered Crop Yield & Price Prediction

Agri-Chain is an end-to-end Machine Learning solution designed to empower farmers by providing data-driven insights. It leverages historical agricultural data to predict crop yields and market prices, helping stakeholders make informed decisions.

## 🚀 Project Overview
- **Objective:** Predict crop productivity and market fluctuations.
- **Backend:** FastAPI (High-performance asynchronous framework).
- **Frontend:** Streamlit (Interactive Dashboard).
- **ML Lifecycle:** Data Cleaning -> EDA -> Feature Engineering -> Model Training -> Deployment.

## 📂 Project Structure
```text
├── data/               # Raw and processed datasets
├── notebooks/          # Jupyter notebooks for EDA and Model Development
├── models/             # Serialized ML models (.pkl / .h5)
├── src/                # Backend API logic (FastAPI)
├── app/                # Frontend UI (Streamlit)
└── requirements.txt    # Project dependencies
```

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib
- **API:** FastAPI, Uvicorn
- **UI:** Streamlit
- **Version Control:** Git & GitHub
